# PTE DI 数据跟读练习

一个专门为PTE考生设计的Describe Image数据跟读工具，帮助快速掌握数字、年份、百分比、温度等各类数据的英文读法。无需注册，完全免费，开箱即用。

## 🎯 设计初衷

作为一名PTE老师，我常在课堂上发现学生在Describe Image部分遇到数据时卡顿、读错或不知如何快速反应。很多练习平台这类功能都开始收费，于是我设计了这款App来帮助我的学生反复跟读各类数据，提升流利度和准确率。今天也把它分享给所有有需要的考生和老师。

## ✨ 主要功能

- **100个随机数据每轮** – 每次加载页面会从庞大的数据池中随机抽取100个数据，涵盖数字、年份、年代、季度、日期、百分比、小数、温度、金额、大数等所有DI常见类型。
- **自动朗读** – 每出现一个新数据自动播放标准英文读法，点击喇叭图标可随时重复播放。
- **计时与进度** – 从第一次点击“下一个”按钮开始计时，进度实时显示（如 34/100），帮助学员掌握练习节奏。
- **完成祝贺** – 完成全部100个数据后弹出提示框，显示本轮所用时间，并立即开始新一轮随机练习，实现无限循环。
- **无需注册** – 打开网页即用，无任何广告或付费墙。

## 🛠️ 技术栈

- **HTML5 / CSS3** – 响应式布局，适配手机、平板、电脑
- **原生 JavaScript** – 无框架依赖，轻量快速
- **Web Speech API** – 浏览器内置语音合成，无需额外资源
- **内嵌数据池** – 所有数据直接定义在脚本中，便于修改和扩展

## ⚙️ 如何使用

1. 打开 `index.html` 或访问部署后的网址。
2. 页面自动显示第一个数据并朗读。
3. 学员大声跟读，然后点击 **“下一个”** 按钮。
4. 重复步骤 2–3，直到完成100个数据。
5. 完成后会弹出恭喜窗口，显示所用时间，并自动开始新一轮随机练习。

> 建议每天练习 2–3 轮，培养数据敏感度和流利度。

## 📝 自定义数据池

如果您想增加或修改数据，请编辑 `script.js` 文件中的 `MASTER_ITEMS` 数组。每个数据对象格式如下：

{ display: "25%", spoken: "twenty-five percent" }
display：页面上显示的内容（可包含逗号、符号等）
spoken：朗读时的文本（应使用英文单词拼写）

您可以根据需要添加任意多条数据，系统会在每次加载时随机抽取100条。
